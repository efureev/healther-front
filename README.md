# Health Check SPA

- Lightning-fast cold server start
- Instant hot module replacement (HMR) and dev SSR
- True on-demand compilation
- Use `vue3 vite4.0 typescript eslint SSR unocss vue-router element-plus scss`

# Getting Started
- spa
```bash 
yarn dev:spa
```
- 
- dev
```bash 
yarn dev
```

- preview
```bash 
yarn preview
```

- build
```bash 
yarn build
```
